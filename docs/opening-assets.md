# British Legends Opening Assets

The opening cinematic loads production background art from `public/assets/opening/`.
Keep these file names stable so the scene data can be replaced without code changes.

## Required Files

| File | Scene Use |
| --- | --- |
| `opening-bibliotheca.png` | Scene 1 and 2: the vast Bibliotheca library |
| `opening-censor.png` | Scene 3: The Censor arrives in black mist |

## Art Direction

Use a dark fantasy JRPG background style with painterly detail, quiet mystery, blue-white magical light, sadness, and a small remaining sense of hope. Each image should be a wide 16:9 game background, no UI, no watermark, no readable logo text except where explicitly requested by the scene.
